# Consent Form Library — Cook County Health

A mobile-friendly web app for physicians and administrative staff to find, manage, and download consent forms. Powered by Smartsheet as the content source.

---

## Features

- **Searchable library** — keyword search by procedure, department, or language
- **Version control** — current approved version clearly shown; retired forms suppressed from search with "Do Not Use" banners
- **Department filters** — filter by Surgery, Hematology, ICU, Ophthalmology, and more (auto-detected from your Smartsheet data)
- **Language filters** — auto-detected from your Smartsheet Language column
- **Department Dashboard** — active forms, review alerts, pending approvals, missing translations, most downloaded
- **Favorites** — staff can heart-save forms per browser
- **Recently used** — tracks which forms a user has opened
- **Most downloaded** — surfaces the most-used forms
- **Direct PDF download** — fetches live download URLs from Smartsheet attachments on demand
- **Mobile-friendly** — works on phones and tablets
